# CodeAlpha FAQ Chatbot

An AI-powered FAQ chatbot built using Python, NLP, and Streamlit.

## Features

- FAQ-based question answering
- NLP preprocessing
- TF-IDF Vectorization
- Cosine Similarity Matching
- Streamlit Interface

## Tech Stack

- Python
- Streamlit
- NLTK
- Scikit-Learn
- Pandas
- NumPy
