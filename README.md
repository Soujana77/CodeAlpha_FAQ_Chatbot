# AI FAQ Assistant using NLP and TF-IDF

## Overview

AI FAQ Assistant is an intelligent question-answering chatbot developed using Python, Natural Language Processing (NLP), TF-IDF Vectorization, Cosine Similarity, and Streamlit.

The chatbot allows users to ask questions related to Artificial Intelligence, Machine Learning, Python Programming, Data Science, Cloud Computing, Cyber Security, Databases, Web Development, and other technical domains. It identifies the most relevant answer from a knowledge base using NLP techniques and displays confidence scores along with related question suggestions.

## Working Process

1. User enters a question.
2. Text is preprocessed using NLP techniques.
3. Questions from the FAQ dataset are vectorized using TF-IDF.
4. Cosine Similarity is calculated between the user query and stored questions.
5. The most relevant answer is retrieved.
6. Confidence score is displayed.
7. Related questions are suggested.
